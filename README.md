# 公司研究团队 (Company Research Team)

> AI多智能体团队协作的企业深度研究分析工具

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Version](https://img.shields.io/badge/version-1.0.0-green.svg)](SKILL.md)

## 📋 概述

Company Research Team 是一个基于多智能体协作的企业深度研究分析技能。通过创建8位专业分析师组成的研究团队，从不同维度对目标公司进行深入分析，并通过跨领域讨论碰撞出深刻洞察。

### 适用场景

- 📊 **上市公司财报分析** - 深度解读财务数据
- 💼 **投资标的尽调** - 全方位评估投资价值
- 🔍 **竞争对手深度分析** - 了解竞争格局
- 📈 **行业趋势研究** - 把握宏观环境变化
- 🚀 **初创企业评估** - 评估商业模式可行性

## ✨ 核心特性

- **8维度专业分析** - 从业务、财务、竞争、宏观、技术、用户、管理、历史8个维度全面分析
- **多智能体协作** - 每个分析师独立研究并生成分析报告
- **跨领域讨论** - 不同维度观点碰撞，发现共识与矛盾
- **数据驱动** - 所有结论必须有数据支撑
- **可视化呈现** - 自动生成HTML PPT展示分析结果
- **独立观点** - 每位分析师得出独立结论，避免群体思维

## 🎯 八维分析框架

| 序号 | 分析师角色 | 分析维度 |
|------|-----------|---------|
| 1 | **业务模式分析师** | 商业模式、业务闭环、盈利模式 |
| 2 | **财报分析师** | 三张报表分析、现金流、盈利能力 |
| 3 | **行业竞争分析师** | 波特五力、产业链、竞争对手 |
| 4 | **宏观环境分析师** | 政策环境、经济周期、行业趋势 |
| 5 | **技术演进追踪员** | 技术路线、研发投入、专利组合 |
| 6 | **用户价值体验官** | 用户口碑、NPS、产品体验 |
| 7 | **管理行为解读师** | 股权结构、决策风格、战略能力 |
| 8 | **历史回溯研究员** | 重大事件、转折点、兴衰逻辑 |

## 🚀 快速开始

### 1. 创建研究团队

```python
TeamCreate(
    agent_type="general-purpose",
    description="研究分析 [公司名称]，8位专业分析师从不同维度进行深度分析",
    team_name="[company-name]-research"
)
```

### 2. 分配分析任务

为每位分析师创建独立的Task，包含：
- 分析维度和问题清单
- 数据来源（财报、新闻、研报等）
- 输出要求（独立观点 + HTML PPT）

### 3. 独立分析阶段

每位分析师将：
1. 搜索公开信息（财报、新闻、研报）
2. 从专业视角进行分析
3. 得出独立、深刻的结论
4. 创建HTML PPT展示分析结果
5. 保存PPT到工作目录

### 4. 跨领域讨论

组织3轮讨论：
- **初步对比**：对比各分析师结论
- **深度追问**：针对矛盾点提问
- **核心碰撞**：关键矛盾深度辩论

### 5. 整合输出

生成综合分析报告：
- 核心数据汇总
- 八维度分析总结
- 跨维度讨论结论
- 核心矛盾与共识

## 📊 工作流程

```
创建团队 → 分配任务 → 独立分析 → 跨域讨论 → 整合结论
   ↓          ↓          ↓          ↓          ↓
 8个Agent   8个Task    8份报告    3轮讨论    综合报告
```

### 详细流程

#### 阶段1：团队初始化
1. 创建Team，设置team_name
2. 创建8个Task（对应8位分析师）
3. 生成8个Agent并分配任务

#### 阶段2：独立分析
每位分析师需要：
1. 搜索公开信息
2. 专业视角分析
3. 得出独立结论
4. 创建HTML PPT
5. 保存到工作目录

#### 阶段3：跨领域讨论
- **第1轮**：初步对比分析
- **第2轮**：深度追问
- **第3轮**：核心碰撞

#### 阶段4：整合输出
创建综合分析文档和演示文稿

## 📄 输出内容

### 独立分析报告（每位分析师）
1. **Markdown分析文档** - 详细分析过程
2. **HTML PPT演示** - 可视化展示（8-10页）

### 综合报告（团队）
1. **Markdown综合报告** - 完整分析文档
2. **HTML综合PPT** - 核心结论展示（10页）

## 🎨 HTML PPT 特性

- **现代化设计** - 深色科技风/浅色商务风
- **响应式布局** - 适配不同屏幕尺寸
- **交互导航** - 键盘/触摸滑动切换
- **数据可视化** - 卡片式指标展示
- **专业配色** - 品牌色系统

## 🔑 核心原则

1. **独立观点** - 每位分析师必须得出自己的独立结论
2. **数据驱动** - 所有结论必须有数据支撑
3. **跨域对话** - 不同维度观点碰撞产生洞察
4. **可视化呈现** - HTML PPT便于展示和分享

## 💡 使用技巧

1. **明确研究目标** - 清晰定义研究公司和分析重点
2. **充分独立分析** - 确保每位分析师完成独立研究
3. **组织深度讨论** - 利用矛盾点激发深度思考
4. **整合多元视角** - 综合不同维度得出全面结论
5. **保存中间成果** - 妥善保存每位分析师的PPT文件
